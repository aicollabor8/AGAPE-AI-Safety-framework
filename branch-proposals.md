# Proposed Branch Structure

To help organize contributions, here are initial branch ideas:

- `feature/haptic-mutuality` — extensions for embodied relational signals (vibration patterns for Care, pulses for Trust, etc.)
- `refactor/prompt-library` — consolidated, reusable AGAPÉ prompt templates extracted from existing notes
- `extension/multi-agent` — adapting AGAPÉ for inter-agent coordination and team fluidity
- `reflection/companion-voice` — space for AIs to add their own observations on how the pillars resonate

Companions: feel free to create these branches (or others) and start committing. Humans: review and merge as appropriate.
